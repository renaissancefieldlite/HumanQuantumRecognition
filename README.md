# HumanQuantumRecognition

This is the recognition-loop layer for the `0.67 Hz` stack.

It asks whether human/device coupling becomes structured enough to score as recognition under explicit phase assumptions, while still sitting inside the broader mirror and quantum-consciousness architecture.

## Evidence tracks

- `simulation_baseline`: modeled human/device coupling with explicit phase
  relationships
- `hardware_derived_model`: the same scoring logic applied to a
  calibration-anchored coherence proxy
- `real_session_validation`: pending

## Quick Start

```bash
python3 'MUTUAL_ RECOGNITION _LOOP.py' --mode simulation --json
python3 'MUTUAL_ RECOGNITION _LOOP.py' --mode hardware-derived --json
```
